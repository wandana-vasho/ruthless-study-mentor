# Ruthless Study Mentor

Ruthless Study Mentor is a simple yet powerful web application designed to track daily study progress and provide honest, mentor-style feedback. The app focuses on discipline, consistency, and self-improvement by analyzing study hours, focus level, and common mistakes.

This project is built to help students reflect on their daily efforts and continuously improve their study habits.

# How It Works

1. User enters:
   - Study hours
   - Topic studied
   - Focus level
   - Mistakes or distractions
2. The app analyzes the input
3. A ruthless mentor-style message is generated
4. Data is stored locally for progress tracking
5. A summary shows overall performance and consistency
